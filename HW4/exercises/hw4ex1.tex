\section*{Exercise 1.}

Let $f$ be an entire function and let $\zeta = e^{2\pi i / n}$ for some $n \in \N$. Suppose that $f(\zeta z) = f(z)$ for every $z \in\C$. Show that there exists an entire function $g$ such that $f(z) = g(z^n)$ for every $z\in \C$.
